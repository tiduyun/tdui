# TDUI

A UI Component Library Based on [element-ui](http://element.eleme.io/#/en-US)

## Installation

```sh
yarn add @tdio/tdui
```

## Contributing and development

Install develop bundler globally

```sh
npm i -g rollup-worker@next
yarn build
```
